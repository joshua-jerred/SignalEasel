## Robot36 quick notes

Audio sample rate of 48000, so 1728000 total samples. This means 7200 samples,
150ms, per scan line.

|                   |                                          |
| ----------------- | ---------------------------------------- |
| *VIS Code*        | 8d (decimal)                             |
| Color Mode        | Y, R-Y, B-Y                              |
| Scan Sequence     | Y, R-Y (even lines) - Y, B-Y (odd lines) |
| Number of Lines   | 240                                      |
| Transmission Time | 36 seconds                               |
| Image Size        | 320x240                                  |
