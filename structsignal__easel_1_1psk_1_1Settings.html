std::string signal_easel::Settings::call_sign {}
inherited

An amateur radio call sign.

Can be left empty if call_sign_mode is set to NONE, but not if using the APRS modulator. Must conform to the FCC's rules for call signs.

Warning
This may be a problem for international users. If so, please create an issue on GitHub.
